Desenvolver uma aplicação web que implementa:

Duas funcionalidades de cadastro: (cadastro de login de usuário, cadastro de alunos, etc)
Duas funcionalidades de edição de cadastros 
Duas funcionalidades de consulta 
Dois tipos de exclusão
A aplicação deve ser responsiva para smartphones e tablets (layout grid em 12 colunas).

A aplicação deve fazer uso de Modals em ao menos uma funcionalidade.

A aplicação deve utilizar outras features do Bootstrap, como carrossel, por exemplo, no front-end.

A aplicação deve ser completamente navegável (não pode ter links quebrados), com o uso de navbars, por exemplo.

A aplicação pode fazer uso de APIs.

Os dados devem ser persistidos em localStorage - não utilizar Banco de Dados.

O tema do projeto é livre e o trabalho pode ser feito em grupo de até 3 pessoas.
